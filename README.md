# 对象分析系统 Love Ops

一个用于记录相亲约会信息的网页应用，支持人选管理、约会记录、关系进展、印象评价、决策计划、日历、统计、搜索与导出等功能。

## 功能概览

- **个人信息管理**：对方资料、年龄职业学历、照片、兴趣爱好、家庭背景、介绍人、认识渠道
- **约会记录**：时间地点、活动内容、花费、感想、照片、时间线展示
- **关系进展**：发展阶段（初识/了解中/交往中/考虑确定关系/已结束）、重要节点
- **印象与评价**：优点、缺点、待观察、标签、性格三观、生活习惯
- **决策与计划**：待确认问题、下次约会计划、是否继续
- **便捷功能**：日历视图、提醒、搜索、导出（Markdown/纯文本）

## 快速开始

```bash
npm install
npm run dev
```

在浏览器打开 `http://localhost:5173` 即可使用。

## 技术栈

- React 18 + TypeScript
- Vite 5
- date-fns
- lucide-react
- localStorage 本地存储（无需后端，可选 Supabase 云同步）
